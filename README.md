# Proyecto Catalogo de películas

## Sobre el proyecto
Proyecto inicial de desarrollo de endpoints para micro servicios usando el framework de FastApi


## Instrucciones de instalación
1. Tener instalado python y git
2. clonar repositorio `git clone https://github.com/sergiogonzalezch/movie-catalog-api `
3. Dirigirse al directorio del proyecto
4. Hacer un entorno `virtual python -m venv venv`
5. Hacer `pip install requeriments.txt`
6. Ejecutar el proyecto con `$ uvicorn main:app --reload`
7. En el navegador ingresar la siguiente URL para entorno local  `http://127.0.0.1:8000/docs`
